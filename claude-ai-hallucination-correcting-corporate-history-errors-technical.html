Section 1: The Thesis

Reddit is selected for AI Overviews because it is dense, query-aligned, and frequently contains stepwise “answer-shaped” language. In a retrieval-based architecture, that makes it a high-scoring candidate even when provenance is weak. The technical problem is not moral. It is structural: the pipeline rewards semantic match and engagement-derived signals, then converts selected candidates into short-form synthesis that amplifies perceived authority.

Stoppage cannot be achieved by a single knob. Quote suppression without citation control leaves Reddit as sanctioned evidence. Citation suppression without eligibility control allows Reddit to re-enter through alternative URLs. Eligibility control without substitution can increase ungrounded synthesis probability. Therefore, a technical stop program requires multiple control surfaces and a measurable closure state: sustained reduction of Reddit grounding events across a controlled query suite and time window, with regression testing after refresh cycles.

Technical premise: if Reddit is in the candidate set, it will reappear under drift. Stop requires eligibility control plus extraction constraints plus monitoring.

Section 2: The Core Analysis

2.1 Pipeline decomposition: where Reddit enters

The pipeline can be decomposed into four stages. Reddit can enter at each stage, and removal at one stage does not guarantee removal downstream.

Stage A — Retrieval Eligibility

Source domains become eligible for candidate generation based on query class, locale, freshness, and policy. If reddit.com is eligible, it will be retrieved for long-tail how-to queries.

Stage B — Reranking & Selection

Candidates are scored by semantic match, perceived utility, and ranking features. “Answer-shaped” threads and high engagement can produce top placement.

Stage C — Citation Assembly

A subset of candidates is assigned as citations. Reddit can remain as evidence even if text is not quoted. This is a sanctioned provenance state.

Stage D — Excerpt & Synthesis

The system extracts spans or synthesizes claims. Reddit influence can appear as verbatim excerpts or paraphrased consensus proxies that obscure UGC origin.

2.2 Event taxonomy: quote, citation, influence

Technical controls require typed events. The relevant event types are:

  • Quote Event: verbatim or near-verbatim Reddit excerpt appears in the overview body.
  • Citation Event: reddit.com URL appears in the overview citation set.
  • Influence Event: the overview paraphrases a Reddit-derived claim as generalized evidence without attribution cues.

2.3 Why Reddit is selection-advantaged

Reddit is selection-advantaged under common retrieval and ranking heuristics:

  • Semantic density: threads contain many phrasings of the same problem, increasing match probability.
  • Answer-like formatting: bullet steps, tool lists, and “here’s what worked” sequences align with how-to intent.
  • Freshness and long-tail coverage: fast emergence for new products, bugs, and edge cases.
  • Engagement features: comments and upvotes can correlate with perceived usefulness, not truth.
  • Entity proximity: users name brands, products, and individuals, increasing retrieval for entity queries.

2.4 Control surface 1: eligibility gating (domain and category)

The highest-leverage control is eligibility gating. If reddit.com is excluded for defined query categories, downstream stages cannot “accidentally” select it. Technical implementations vary, but the functional requirement is consistent: domain eligibility is conditional on query class and entity sensitivity. The strictest posture is a hard ban for high-impact categories and entity-sensitive queries. A softer posture applies a strong penalty and quota caps, paired with authoritative substitution.

2.5 Control surface 2: reranking penalties and source quotas

If Reddit remains eligible, reranking and quotas are required. Reranking penalties reduce selection likelihood by domain and by thread type. Quotas limit the maximum number of UGC sources in evidence slots. Quotas must be applied before citation assembly, not after, to prevent “citation laundering” where Reddit remains as sanctioned provenance but is not directly quoted.

2.6 Control surface 3: citation assembly constraints

Citation assembly must enforce domain rules independently of retrieval and reranking. Otherwise, Reddit can enter via alternate pathways (e.g., if a candidate survives due to high semantic score). Citation constraints include:

  • Domain exclusion lists: Reddit prohibited in specified categories.
  • Evidence slot typing: high-impact claims require higher-provenance sources.
  • Source diversity constraints: prevent multiple UGC sources from dominating citations.

2.7 Control surface 4: excerpt filtering and paraphrase constraints

Stopping “quoting” requires excerpt filtering: block verbatim spans from Reddit even if cited. However, stopping influence requires more: paraphrase constraints. The system must avoid converting UGC anecdote into generalized statements that read as verified. A minimal constraint is to preserve uncertainty markers when the only support is UGC. A stricter constraint is to abstain from generating claims that would require authoritative evidence.

2.8 Integrity constraint: substitution or abstention

Removing Reddit without substitution can increase ungrounded synthesis. Therefore, technical stoppage requires an integrity rule:

  • Substitution: require authoritative candidates to fill evidence slots when UGC is excluded.
  • Abstention: suppress the overview when substitution fails or when evidence quality is below threshold.
A stop program that increases hallucination rate is a control failure. Removal must not trade one hazard for another.

2.9 Measurement: define “stop” as an auditable state

Stoppage is verified by rates, not anecdotes. Minimum metrics include:

  • Reddit citation rate: percentage of overviews with reddit.com in citations.
  • Reddit quote rate: percentage of overviews with verbatim/near-verbatim spans attributable to Reddit.
  • Influence rate: percentage of overviews where core claims track to Reddit without authoritative support.
  • Category-conditioned rates: rates computed separately for high-impact categories and entity-sensitive queries.
  • Regression drift: rate deltas after index/ranking refresh events.

2.11 Stop criteria (technical closure)

Technical closure requires declared thresholds and a defined window. Typical closure assertions are:

  • Stop-Q: quote event rate equals 0 across the suite and window.
  • Stop-C: citation event rate is below threshold T (often 0 for high-impact categories).
  • Stop-I: influence rate is below threshold T, validated by traceability checks.

Closure requires regression revalidation after refresh cycles. Without regression, stoppage is temporary by definition.

Section 4: The Vance Protocol

The Vance Protocol defines a technical stop program with explicit event typing, layered controls, and measurable closure. It treats Reddit as a controllable exposure and defines “stop” as a sustained reduction of grounding events across a fixed suite and window, with regression testing after refresh cycles. The protocol requires integrity: authoritative substitution or abstention when UGC is excluded.

4.1 Protocol steps (technical execution)

  • Step 0 — Type events: quote, citation, and influence are tracked separately.
  • Step 1 — Baseline: compute event rates across a fixed query suite and locale matrix.
  • Step 2 — Gate eligibility: domain/category rules for reddit.com by risk class.
  • Step 3 — Constrain citations: quotas and domain constraints before evidence slot finalization.
  • Step 4 — Filter extraction: prohibit verbatim excerpts; constrain paraphrase to prevent consensus proxy.
  • Step 5 — Enforce integrity: substitution or abstention to prevent ungrounded synthesis.
  • Step 6 — Verify window: sustained compliance across the defined time window.
  • Step 7 — Regression: rerun after refresh epochs and document drift.

4.2 Closure criteria

Closure is permitted only when typed event rates remain below declared thresholds across the declared window and when post-refresh regression testing confirms durability. Any other closure is temporary by definition.
